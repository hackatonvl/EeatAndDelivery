Задача: сделать приложение для суши-кухни и службы доставки. Приложение автоматизирует процесс. Получает заказы с сайта. Менеджер обрабатывает заявку, возможно добавляет комментарий. Переводит заказ на кухню. Повара готовят и переводят заказ в службу доставки.

Демка приложения тут: http://tnebo:3000
